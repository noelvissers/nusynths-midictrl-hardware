# NU Synths | MIDICTRL - Hardware

Hardware design of the NU Synths 'MIDICTRL' eurorack module.

For firmware see [nusynths-midictrl](https://github.com/noelvissers/nusynths-midictrl) repository.

## Contents

The following files and folders are in this repository:

|File / Folder           |Remarks                       |
|------------------------|------------------------------|
|midictrl-assembly       |Assembly document of all PCBAs|
|midictrl-connector-board|Connector board assembly files|
|midictrl-main-board     |Main board assembly files     |
|usb-c-breakout-board    |USB type C breakout board     |

## Dependencies

Altium Designer or Altium Designer Viewer
